# Contributing to TAOP

Thanks for your interest in TAOP — the on-chain credit bureau and capability registry for AI agents.

## Getting Started

1. Clone the repo
2. `npm install`
3. `npm run contracts:build`
4. `npm run contracts:test`

See `README.md` for running the full stack (backend + demo).

## Development

- Contracts are in `contracts/`
- Tests: `test/`
- Backend: `packages/backend/`
- SDK: `packages/sdk/`
- Python: `packages/python-sdk/`

Run type checking: `npm run typecheck`

## Pull Requests

- Keep PRs focused.
- Update tests for contract changes.
- Run `npm run contracts:test` and typecheck before submitting.
- Link to relevant issues or the `IMPROVEMENTS_PLAN.md`.

## Code of Conduct

Be respectful. This is early infrastructure for the agent economy.

## Questions?

Open an issue or start a discussion on the repo.
